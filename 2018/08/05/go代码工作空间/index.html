go代码工作空间

go代码必须放置在工作空间内，它其实就是一个目录，其中包含三个子目录。

1. src 目录

目录包含Go的源文件，它们被组织成包（每个目录都对应一个包）

2. pkg 目录

pkg 目录包含包对象

3. bin 目录

bin目录包含可执行命令

4. 将bin目录加入系统path

这样在任何地方都可执行go可执行文件
